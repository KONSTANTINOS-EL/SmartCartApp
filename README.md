# SmartCartApp

Smart cart management with Python (Flask), MongoDB and Streamlit UI.

# Description


**SmartCartApp** is a shopping cart management platform with a backend in Python Flask, a MongoDB base and a frontend in Streamlit.

# Details

- Python 3.8
- MongoDB 
- pip
